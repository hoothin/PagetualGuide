# action
>0은 URL을 로드하고 정적 HTML을 삽입하는 것을 의미하며, 1은 동적 JavaScript 코드가 페이지에 영향을 미칠 수 있도록 iframe으로 로드하는 것을 의미하고, 2는 iframe을 하단에 강제로 삽입하는 것을 의미합니다.

Example:
--
```json
"action": 1
```

+ action: 0
다음 링크의 URL을 가져와 동적 처리 없이 정적 HTML을 가져옵니다. 이는 기본 동작이며 가장 빠른 동작입니다.
+ action: 1
URL을 iframe에 로드하고 로딩 완료 후 콘텐츠를 가져옵니다. 이는 페이지 열린 후 콘텐츠가 로드되는 상황에 사용됩니다.
+ action: 2
다음 링크의 전체 페이지 iframe을 생성합니다. 페이지를 다듬어 현재 페이지에 삽입합니다.
