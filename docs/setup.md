::: details ğŸ’¡ Q: What is "Browser Greasemonkey script"?
A: "Greasemonkey" is a plug-in that runs and manages scripts on the browser. A script is a piece of code that optimizes your web browsing experience. Once installed, some scripts add new functionality to the site, some make the site's interface easier to use, and some hide annoying ads on the site.
:::

::: details ğŸ’¡ Q: Why is Pagetual written as a userscript instead of an extension?
A: Because Pagetual has powerful custom code execution capabilities to implement paging rules, and extensions have restrictions on custom code execution capabilities for security reasons. In addition, userscripts are more flexible and can run on almost any operating system, such as Windows/MAC/iOS/Android.
:::

::: details ğŸ’¡ Q: Are user scripts safe?
A: Userscripts themselves are not malicious, but security depends on the scripts you choose to install. It is recommended to only obtain scripts from trusted sources and read the script code carefully. And of course, you can trust me - I've been contributing to the open-source community for ten years.
:::

::: details ğŸ’¡ Q: How to install user script?
A: First, you need to install a script manager plug-in (such as Tampermonkey) on your browser. You can then install and manage userscripts in the plugin management interface.
:::

::: details ğŸ’¡ Q: Can I use userscripts on mobile devices?
A: Typically, the default browser of mobile devices does not support user scripts. However, some specific browsers provide plug-ins or extensions with related functions.
:::

::: details ğŸ’¡ Q: How to disable or delete userscripts?
A: You can disable or delete specific Grease Monkey scripts in the user script plug-in management interface. You can also temporarily disable the entire Script Manager plugin to stop all scripts from running.
:::
