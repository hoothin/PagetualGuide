# 轻松上手，玩转“永页机”

# 基础操作：像呼吸一样自然

装好了？太棒了！现在就去百度open in new window搜点东西试试，然后向下滚动鼠标……看到了吗？下一页就这么神不知鬼不觉地出现了！

# 探索“翻页隔条”的秘密

每次翻页，你会看到一个“隔条”出现，它可是个多功能宝藏！

  • 左边的太极 ☯️： 一键“返回顶部”，瞬间回到页面之巅。
  • 右边的太极 ☯️： 一键“直达底部”，体验飞一般的穿梭。
  • 中间的页码： 它不仅告诉你这是第几页，点一下还能“暂停翻页”，让你稍作喘息。
  • “任意门”跳转： 如果网站支持，点一下页码数字，输入你想去的页数，嗖~ 一下就到！
  • 追求极致？隐藏它！ 在配置页，你可以让隔条“隐身”，配合侧边栏，享受真正的“无缝沉浸式”阅读。

# 解锁“命令菜单”的超能力

在浏览器右上角找到脚本管理器（比如 Tampermonkey）的图标，点一下，找到“东方永页机”，你会发现几个隐藏的“超能力”：

  • 打开配置页： 这里是你的“秘密基地”，所有个性化设置和自定义规则都在此进行。
  • 在此站禁用翻页： 某个网站不希望自动翻页？点它，给这个网站“放个假”。
  • 强制拼接： 遇到“永页机”偶尔“犯迷糊”的网站？用这个“杀手锏”！它会用最强硬的手段，把下一页的内容“抓”过来。
  • 更新在线规则： “永页机”默认每两天会自动“修炼”一次，获取最新的网站规则。你也可以点这里，让它立刻“功力大增”！

# 打造你的专属“永页机”

进入“配置页”，开始你的“私人订制”之旅吧！

  • 写入历史记录： 强迫症福音！开启后，每次翻页都会留下“足迹”。下次从链接点进去再返回，能直接回到上次看到的地方。
  • 新标签页打开： 勾选它，所有在拼接页面里新打开的链接，都会乖乖地在新标签页中出现。
  • 预读下一页： 像个预言家！提前加载下一页的图片，让你在翻页时快人一步，告别等待。
  • 导入“武功秘籍”： 你可以导入各路大神分享的“规则链接”，也可以自己动手，丰衣足食。这些规则还支持排序和“热更新”，随时保持最佳状态！

# 终极挑战：创造你的第一条规则！

准备好成为“规则大师”了吗？别怕，我们有“可视化抓取”神器！

  1. 开启“抓取”模式： 点击“自定义规则”，你的鼠标会变成一个蓝色选框。
  2. “选中”你想要的： 在页面上，对着你希望无限加载的主体内容（比如文章列表、图片瀑布流），点一下！“永页机”会自动帮你转换成代码。
  3. 微调与确认： 选错了？没关系！在弹出的菜单里可以轻松切换到父节点。你也可以手动修改代码，点击“√”还能立刻检查对不对。
  4. 开始编辑： 一切就绪？点击铅笔图标 ✏️，进入配置页，你的第一条规则雏形已经在那儿等着你了！

你的第一条规则长这样：

  {
    "name":"我的第一条规则",             // 给它起个响亮的名字
    "url":"^https://www.example.com/",  // 在哪个网站生效
    "pageElement":"div.content"         // 你想让哪部分内容无限加载
  }

是不是很简单？你还可以添加更多高级选项，比如“下一页按钮在哪”、“隔多久翻一次页”等等，慢慢探索吧！

小贴士：

  • 拖拽菜单标题可以移动窗口。
  • 点击标题栏左边的太极图标可以暂停“抓取”模式。
  • 点击下方的 🔽 图标可以开启“快捷编辑”，快去试试吧！
