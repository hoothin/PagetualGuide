Pagetual

Most compatible Auto pager script ever! Auto loading paginated web pages for 90% of all sites!

Get started →

brevity comes first

The least intrusive automatic page-turning solution that helps you focus on web browsing with minimal operations.

self-propelled

Enjoy intelligent self-driven page turning, find the next page, and extract content automatically. After the site is revised, you don't need to wait for the rules to be updated.

high performance

Pagetual is compatible with 3800+ rules of wedata, and has done a lot of performance optimization, while preparing powerful custom rules for advanced users.
