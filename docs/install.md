Don't have one yet? No problem! Just pick one from the list below for your browser. **Tampermonkey** is the most popular choice and a great place to start.

### Frequently Asked Questions

::: details üí° What even *is* a userscript?
A: Think of it as a mini-extension! A userscript is a small piece of code that customizes a specific website. Some add new features, others make sites easier to use, and some (like Pagetual) fundamentally improve your browsing experience.
:::

::: details üí° Why a userscript and not a regular extension?
A: Great question! Pagetual has some powerful features that require more flexibility than standard browser extensions are allowed. Userscripts give us that flexibility and have the awesome benefit of working on almost any platform, including Windows, macOS, iOS, and Android.
:::

::: details üí° Is this safe to install?
A: The script itself is perfectly safe. However, just like with any software, you should always be careful about where you get your scripts from. We recommend sticking to trusted sources like Greasyfork (our primary download link) and our official GitHub page. The code is open for anyone to review!
:::

::: details üí° How do I turn it off or remove it?
A: You're always in control! You can easily disable or delete any userscript from your script manager's dashboard. You can also temporarily disable the entire script manager to turn everything off at once.
:::
