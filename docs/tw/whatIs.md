# 東方永頁機：最強自動翻頁腳本，自動載入並拼接下一頁
東方永頁機是一款免費的超絕推薦瀏覽器擴充功能使用者油猴腳本，可自動載入網頁的下一頁。

它使用 JavaScript 自動分析網頁的結構，找到下一頁的鏈接，並加載內容之後將其插入到當前頁面中。 這使您可以輕鬆瀏覽網頁，而無需單擊“下一頁”按鈕。

東方永頁機適用於大多數網站，無需任何規則。
