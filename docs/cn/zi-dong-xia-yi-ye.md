# 最强自动翻页插件
东方永页机是一款免费的浏览器扩展用户脚本，可自动加载并拼接网页的下一页。

它使用 JavaScript 自动分析网页的结构，找到下一页的链接，并加载内容之后将其插入到当前页面中。这使您可以轻松地浏览网页，而无需单击“下一页”按钮。

东方永页机适用于大多数网站，无需任何规则。
