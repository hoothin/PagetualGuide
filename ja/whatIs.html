# 東方永頁機とは何ですか?

東方永頁機は、AutoPagerize 代替、最強の Web ページの次のページを自動的にロードする無料のブラウザ拡張機能のユーザー スクリプトです。

WEB サイトのページ分割問題を解決する。自動的にページを結合し無限スクロールのような体験が可能。

JavaScript を使用して、Web ページの構造を自動的に分析し、次のページへのリンクを見つけて、コンテンツを読み込んだ後に現在のページにコンテンツを挿入します。 これにより、「次のページ」ボタンをクリックすることなく、簡単に Web を閲覧できるようになります。

東方永頁機はほとんどの Web サイトでルールなしで動作します。

# 機能と利点

  • 自動ロード:

東方永頁機を使用すると、手動で行うことなく、Web ページの次のページを自動的に読み込むことができます。

  • ルールは必要ありません:

東方永頁機は、ルールを設定することなく、ほとんどの Web サイトのページ構造に自動的に適応できます。

  • 豊富なカスタマイズ設定:

東方永頁機には、ニーズに応じて調整できる豊富な設定オプションが用意されています。

# 使用するシーン

  • 長い Web ページを閲覧する

東方永頁機を使用すると、ページの一番下までスクロールすることなく、長い Web ページを簡単に閲覧できます。

  • ニュースサイトを読む

東方永頁機 を使用すると、「次のページ」ボタンをクリックせずに、ニュース Web サイトの記事を簡単に読むことができます。

  • フォーラムの投稿を表示

東方永頁機を使用すると、ページをめくることなく、フォーラムの投稿に対するすべての返信を簡単に表示できます。

  • 写真サイトを閲覧する

東方永頁機を使用すると、「次へ」ボタンをクリックすることなく、写真 Web サイトのすべての写真を簡単に閲覧できます。

# 設置環境

Chrome、Firefox、Edge、Opera、Safari などのブラウザと Android / iOS プラットフォームをサポートします。
