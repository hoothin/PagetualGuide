# Devenez un sponsor de Pagetual
**Pagetual** est un projet open source sous licence MPL-2.0 et est entièrement gratuit pour tous. Mais si nous voulons que tout le monde profite de la commodité apportée par **Pagetual**, nous devons maintenir des milliers de règles pour nous adapter à d'innombrables pages sur Internet. De plus, de nouvelles fonctions sont continuellement développées pour rendre **Pagetual** plus parfait. Ces efforts nécessitent un financement et une main-d'œuvre importants, c'est pourquoi nous espérons votre généreux soutien financier.

## Comment sponsoriser
Vous pouvez sponsoriser via [Ko-fi](https://ko-fi.com/hoothin) ou [afdian](https://afdian.com/a/hoothin). Les parrainages mensuels récurrents et les dons uniques sont acceptés. Les parrainages récurrents donnent droit à des placements de logo comme spécifié dans les Niveaux de Parrainage.

## Parrainer Pagetual en tant qu'entreprise
Parrainer **Pagetual** vous permet d'atteindre plus de 200 000 utilisateurs de qualité, curieux et aventureux, à travers le monde via notre site web et les fichiers readme de nos projets GitHub. De plus, soutenir des projets open source non seulement élargit votre portée, mais contribue également à améliorer la réputation de votre marque, ce qui est un atout important pour toute entreprise.

Si vous développez un produit dont les clients cibles sont des développeurs ou qui présente une certaine courbe d'apprentissage, vous obtiendrez un trafic de qualité grâce à une exposition sponsorisée, car tous nos visiteurs sont des utilisateurs de qualité dotés d'excellentes compétences pratiques. Vous pouvez attirer un groupe d'utilisateurs intéressés par votre produit et désireux de le mettre en pratique. En outre, les parrainages peuvent vous aider à renforcer la notoriété de votre marque et à augmenter les taux de conversion.

## Parrainer Pagetual en tant que particulier
Si vous êtes un utilisateur individuel et que vous êtes satisfait de la commodité apportée par l'utilisation de **Pagetual**, veuillez envisager d'exprimer votre gratitude sous la forme d'un don, comme si vous nous offriez une tasse de café de temps en temps. Beaucoup de nos contributeurs reçoivent des parrainages et des dons via GitHub. Vous pouvez trouver le bouton Sponsor sur le profil de chaque membre sur la [page de l'équipe du projet](https://github.com/hoothin/UserScripts/graphs/contributors) pour plus de détails. Nous apprécions sincèrement votre soutien et votre générosité.

## Avantages des niveaux
+ Or (10 USD/mois) :
   - Placer le logo dans une position visible sur la page d'accueil de **Pagetual**.
   - Placer votre logo en évidence dans la barre latérale de toutes les pages de contenu.
   - Placer le logo en évidence dans le README sur Github.
+ Argent (5 USD/mois) :
   - Placer le logo sur la page d'accueil de **Pagetual**.
   - Placer le logo dans le README sur Github.
+ Bronze (2 USD/mois) :
   - Placer le surnom dans le README de Github.

Si vous avez des questions sur les niveaux, le processus de paiement ou les données d'exposition du parrainage, veuillez contacter [rixixi@gmail.com](mailto:rixixi@gmail.com).
