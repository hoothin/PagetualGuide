::: details 💡 Q：什麼是“油猴”？
A：“油猴”是一種在瀏覽器上運行和管理腳本的插件。所謂腳本就是一段代碼，它們能夠優化您的網頁瀏覽體驗。安裝之後，有些腳本能為網站添加新的功能，有些能使網站的界面更加易用，有些則能隱藏網站上煩人的廣告。
:::
