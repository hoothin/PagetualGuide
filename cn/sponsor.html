# 成为东方永页机赞助者

东方永页机 是 MPL-2.0 许可的开源项目，完全免费提供给所有人使用。 但是想让所有人都能享受到 东方永页机 带来的便利，就需要维护成千上万条规则，以适配互联网中不计其数的页面。 此外，不断开发的新功能，让 东方永页机 更加完善。 这些工作都需要大量的资金和人力，因此我们希望得到您慷慨的财政支持。

# 如何赞助

您可以通过 爱发电open in new window 或 Ko-fiopen in new window 进行赞助。我们接受每月定期赞助和一次性捐赠。 定期赞助将可获得赞助级别中指定的徽标位置。

# 企业赞助

赞助 东方永页机 可让您通过我们的网站和 GitHub 项目自述文件触及全球逾 20 万充满好奇心和冒险精神的优质用户。 此外，支持开源项目不仅能够扩大您的影响范围，还有助于提升您的品牌声誉，这对于任何公司而言都是一项重要的资产。

如果您正在构建一个目标客户是开发人员或者有一些学习曲线的产品，您将通过赞助曝光获得高质量的流量，因为我们所有的访问者都是具有出色动手能力的优质用户。 您能够吸引到对您产品感兴趣且愿意付诸实践的用户群体。 此外，赞助还能够帮助您建立品牌知名度，并提高转化率。

# 作为个人赞助

如果您是个人用户，并且对于使用 东方永页机 带来的便利感到满意，请考虑以捐赠的方式表达您的感激之情，就像偶尔为我们买杯咖啡。 我们的许多贡献者通过 GitHub 接受赞助和捐赠。您可以在项目团队页面open in new window上的每个成员个人资料中找到“赞助”按钮，以了解更多详情。 我们由衷地感谢您的支持和慷慨。
