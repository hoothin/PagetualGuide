# pinUrl
>Иногда nextLink или pageElement отсутствует. Установите true, чтобы привязать правило только по url без интеллектуального поиска.

Пример:
--
```json
"pinUrl": true
```
>Когда URL соответствует заданному RegExp, правило применяется, даже если элементы не совпадают.

*Например, если nextLink появляется только после прокрутки или клика, используйте pinUrl=true.*
