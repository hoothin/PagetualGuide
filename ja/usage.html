# 使い方ガイド

# 1. 基本的な使い方

まずは試してみよう！ Yahooopen in new window を開いて、ぐーっと下までスクロールしてみて。次のページが自動でくっついてくれば成功だよ！

# 💡 ページ区切りバーの便利な機能

ページが自動で切り替わる時、間に便利な「区切りバー」が表示されるんだ。これ、ただの飾りじゃないんだぜ。

  • 左の☯️ (太極図) アイコン: ページの先頭へ一瞬で戻れる！
  • 右の☯️ (太極図) アイコン: ページの末尾へ一瞬でジャンプ！
  • 真ん中の「ページX」: クリックすると、次のページへのリンクとして機能するよ。

# ✨ 没入モードで、もっと快適に

設定ページで区切りバーを非表示にできるよ。これを隠してサイドバーを追加すれば、まるで1つの長いページを読んでるみたいに、シームレスな体験ができるんだ。

# 🔢 ページ番号ジャンプ

サイトのルールにpageNum項目があれば、区切りバーの数字をクリックすると入力ボックスが出現。好きなページ番号を入れれば、そのページへ直接ジャンプできる！

# ⏸️ ちょっと休憩

区切りバーのどこかをクリックすれば、自動ページめくりを一時停止できるよ。

# 🔧 コマンドメニューを使いこなそう

ブラウザのTampermonkeyアイコンをクリックすると、Pagetual (永頁機) のメニューが出てくるよ。

# ⚙️ 設定ページを開く

クリックすると設定ページへ。見た目の調整や、後述する最強の「カスタムルール」作成はここから！

# ❌ このサイトでは自動めくりをOFFにする

「このサイトではいらないな…」って思ったら、これをクリック。今見ているサイト全体で、Pagetualを一時的に無効にできるよ。

# 💥 強制読み込みモード

賢い自動ルールがたまに間違えちゃうことがある。そんな時はコレ！ スクリプトが強制的にページ全体を読み込もうと頑張ってくれる。最後の手段として使ってみて。

# 🔄 オンラインルールを今すぐ更新

Pagetualは2日ごとに自動でルールを更新してくれるけど、「今すぐ最新のルールが欲しい！」って時はこれをクリックしてね。

# 🎨 自分好みにカスタマイズ

設定ページでは、区切りバーの透明度を変えたり、閲覧履歴の記録方法を変えたり、リンクを新しいタブで開くようにしたり…いろんな設定が君を待っている。

他の人が作ったルールをインポートしたり、自分でルールを直接書き込んだりもできる。ルールの順番を入れ替えたり、特定のルールだけを即時更新したり、自由自在だ！

# 🕒 閲覧履歴を記録する

これをオンにすると、ページをめくるたびにURLが履歴に残るようになる。だから、一度閉じてしまっても、ブラウザの「戻る」で前回見ていた場所の近くまで戻れるんだ。地味に便利！

# ↗️ リンクを新しいタブで開く

これをオンにすると、自動で読み込まれたページ内のリンクは、全部新しいタブで開かれるようになるよ。

# 🚀 次のページを先読みして高速化

これをオンにすると、君が今のページを読んでいる間に、裏側で次のページの画像をこっそり読み込んでくれる。だから、表示がめちゃくちゃ速くなる！

# 🚀 最強の武器！カスタムルール作成 (上級者向け)

「このサイト、自動で動かない…！」そんな時は、君自身がルールを作るんだ！

# 🖱️ 見たままルール作成

カスタムルール作成ボタンを押すと、ページ上に青い選択ボックスが現れる。あとは、自動で読み込みたいメインコンテンツの部分をクリックするだけ！自動でコードに変換してくれるんだ。

# 👉 セレクターを微調整

メインの要素を選んだ後、コードにマウスを合わせると、その親要素たちが見えるようになる。クリックして、もっとピッタリな範囲に切り替えよう。もちろん、手動でセレクターを書き換えて「√」を押せば、すぐに適用範囲を確認できるよ。

# ✍️ 編集スタート

準備ができたら、鉛筆アイコンをクリックして設定ページへ。ここから本格的なルールの編集が始まる。

# 🔰 最初のカスタムルール

例えば、こんな感じのシンプルなルールから始めよう。

   {
     "name":"beauty", // ルール名 (なんでもOK)
     "url":"^https://www\.beautyxxx\.com/", // ルールを適用したいページのURL
     "pageElement":"p>img" // 次のページから持ってくる要素
   }

他にも「次のページボタンの場所」や「ページの読み込み間隔」など、いろんな項目を追加して、君だけの最強ルールを作り上げよう！

# その他のヒント

  • ウィンドウの移動: メニューのタイトルバーをドラッグすれば、好きな場所に動かせるよ。
  • 一時停止: タイトルバーの左にある☯️アイコンをクリックすれば、見たままルール作成を一時停止できる。
  • クイック編集: 下の🔽アイコンをクリックすると、簡易編集モードに。上で確認したセレクターを、対応する項目にサッと貼り付けて保存！
