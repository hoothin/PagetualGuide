# Fonctionnalités

  • Chargement automatique :

Pagetual peut charger automatiquement la page suivante d'une page web sans que vous ayez à le faire manuellement.

  • Aucune règle requise :

Pagetual peut s'adapter automatiquement à la structure de page de la plupart des sites web sans que vous ayez à définir de règles.

# Scènes d'utilisation

  • Parcourir de longues pages web :

Pagetual vous permet de parcourir facilement de longues pages web sans avoir à faire défiler jusqu'en bas de la page.

  • Lire des sites d'actualités :

Pagetual vous permet de lire facilement des articles de sites d'actualités sans cliquer sur le bouton "Page suivante".

  • Afficher les publications de forum :

Pagetual vous permet de visualiser facilement toutes les réponses à une publication de forum sans tourner les pages.

  • Parcourir les sites d'images :

Pagetual vous permet de parcourir facilement toutes les images d'un site web d'images sans avoir à cliquer sur le bouton "suivant".
